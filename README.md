# 贪吃蛇
Python实现的贪吃蛇

## 要求
* evdev
* Linux
* Python 2.7

## 用法
```Shell
sudo python ./snake.py
```
* 注意: 为了获取键盘事件, 脚本必须在sudo模式下运行; detectInputKey()函数默认从/dev/input/event4获取输入, 实际情况需要根据自身电脑调整
